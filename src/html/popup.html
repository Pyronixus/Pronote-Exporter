ℹ️ À propos

Cette extension exporte automatiquement vos notes PRONOTE depuis votre navigateur.

  • JSON : Format structuré universel
  • CSV : Pour Excel et feuilles de calcul
  • Markdown : Pour documentation et wikis
⚠️ Important

• Les données sont exportées depuis votre navigateur
• Aucune donnée n'est envoyée à des serveurs
• Ouvrez la console (F12) si pas de données trouvées
